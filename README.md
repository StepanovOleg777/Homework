# ПРОЕКТ "МОЙ ПРОЕКТ PYTHON"

## Описание:

 ПРОЕКТ "МОЙ ПРОЕКТ PYTHON" - это приложение для работы с 
банковскими операциями.

## Установка:

1.Клонируйте репозиторий:
```
git clone https://github.com/StepanovOleg777/Homework.git
```
2. Установите зависимости:
```
pip install -r requirements.txt
```
## Тестирование:

 Проект покрыт тестами Pytest. Для запуска выполните команду:
 ```
Pytest
```
