# ПРОЕКТ "МОЙ ПРОЕКТ PYTHON"

## Описание:

 ПРОЕКТ "МОЙ ПРОЕКТ PYTHON" - это приложение для работы с 
банковскими операциями. В проект добавлены функции по считыванию
операций из CSV и EXCEL файлов.Пользователь может отфильтровать
необходимые операции по дате, валюте, а так же по статусу операции.

## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/StepanovOleg777/Homework.git
```
2. Установите зависимости:
```
pip install -r requirements.txt
```
## Тестирование:

1. Проект покрыт тестами Pytest. Для запуска выполните команду:
 ```
Pytest
```
2. В проекте так же используется логирование модулей.
