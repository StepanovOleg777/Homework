# ПРОЕКТ "МОЙ ПРОЕКТ PYTHON"

## Описание:

 ПРОЕКТ "МОЙ ПРОЕКТ PYTHON" - это приложение для работы с 
банковскими операциями (счетами и картами). В проект добавлены функции по считыванию
операций из CSV, JSON и EXCEL файлов.В проекте так же использована библиотека
Logging, которая позволяет записывать сообщения в лог-файлы, чтобы отслеживать работу приложения.

## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/StepanovOleg777/Homework.git
```
2. Установите зависимости:
```
pip install -r requirements.txt
```
3. Создание виртуального окружения

```
python3 -m venv venv
```
4. Установка зависимостей

```
pip3 install -r requirements.txt
```


## Тестирование:

 Проект покрыт тестами Pytest. Для запуска выполните команду:
 ```
Pytest
```
